# Mono

A minimal focus app for managing your daily tasks.

## Features

- **Add tasks fast** — Just start typing
- **Mark tasks urgent** — Press Cmd+U before adding a task to flag it as high priority
- **Toggle done/undone** — Click any task to mark it complete, click again to undo
- **Delete tasks** — Click the delete button to remove a task
- **Live clock** — Displays the current date and time at the top
- **Persistent storage** — Your tasks are saved to local storage automatically

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to start.

## Build & Deploy

```bash
npm run build
npm run deploy
```
